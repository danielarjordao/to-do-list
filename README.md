# To-Do List

Um aplicativo web simples e prático para gerenciar suas tarefas do dia a dia.

## Estrutura do Projeto

- **index.html**: Página principal do aplicativo.
- **style.css**: Estilos utilizados no site.
- **script.js**: Lógica e funcionalidades do aplicativo.

## Como visualizar

Você pode acessar o aplicativo diretamente pelo GitHub Pages:

[https://danielarjordao.github.io/to-do-list/](https://danielarjordao.github.io/to-do-list/)

Ou, se preferir, clone o repositório e abra o arquivo `index.html` em seu navegador.
